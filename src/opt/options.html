Text Contrast for Dark Themes Options

Contrast Threshold (Beta)

Minimum contrast required for recoloring elements. Extension only re-colors elements if resulting contrast is higher than this threshold. Meaningless if not using "Use system colors" option.

Delay (Beta)

Optional delay for operation of extension. This may be needed if another style-changing extension (e.g. stylish, dotjs, dark mode) randomly interferes. The extension will delay itself by the specified amount to ensure that other style changes occur before processing. Usually a few 10s of ms is sufficient. Set to 0 to disable delay.

ms

Override Lists

Ideally, this extension shouldn't break sites, so if you find yourself adding a site to the 'Disable extension' section to deal with an issue, consider filing a bug at https://github.com/DmitriK/darkContrast/issues

Type a match string into the text boxes, one per line. Press 'Save' when done. Extension will do simple matching against the site URL.
